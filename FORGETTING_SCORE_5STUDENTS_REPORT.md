## 📊 Forgetting Score 分析

### 使用的公式

```
F_c(t) = (1 - s_{t,c}) × (Δt_c / (Δt_c + τ))

其中:
- s_{t,c}: 预测的正确概率（基于历史正确率估算）
- Δt_c: 距离上次做该concept的时间间隔（分钟）
- τ = 3天 = 4320分钟
```

### 优先级阈值

- 🔴 **紧急** (FS ≥ 0.3): 需要立即复习
- 🟠 **重要** (0.2 ≤ FS < 0.3): 本周内复习
- 🟡 **一般** (0.1 ≤ FS < 0.2): 下周复习
- 🟢 **维持** (FS < 0.1): 维持性复习
