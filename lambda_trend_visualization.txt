关键观察
================================================================================

1. GPT趋势（G）：
   ┌─────────────────────────────────────────────────────────┐
   │ 0.0 → 0.1: ↓ 下降                                       │
   │ 0.1 → 0.5: ↑ 逐步上升                                   │
   │ 0.5 → 0.9: ↓ 波动下降                                   │
   │ 0.9 → 1.0: ↑↑ 显著上升至峰值 (0.5952)                   │
   └─────────────────────────────────────────────────────────┘
   结论: GPT偏好完全记忆策略，在λ=1.0时表现最佳

2. QWEN趋势（Q）：
   ┌─────────────────────────────────────────────────────────┐
   │ 0.0 → 0.7: ↑ 缓慢上升                                   │
   │ 0.7: ⭐ 达到峰值 (0.5967)                               │
   │ 0.7 → 1.0: ↓ 显著下降                                   │
   └─────────────────────────────────────────────────────────┘
   结论: QWEN需要适度遗忘，完全记忆反而降低性能

3. LLAMA趋势（L）：
   ┌─────────────────────────────────────────────────────────┐
   │ 0.0 → 0.1: ↑ 上升                                       │
   │ 0.1 → 0.3: ↓ 下降                                       │
   │ 0.3 → 0.5: ↑↑ 快速上升至峰值 (0.5919)                   │
   │ 0.5 → 0.7: ↓ 下降                                       │
   │ 0.7 → 1.0: ↑ 恢复上升                                   │
   └─────────────────────────────────────────────────────────┘
   结论: LLAMA在中等记忆保留(λ=0.5)时达到最佳平衡

================================================================================
性能排名
================================================================================

总体最佳配置（按Avg Learning Gain排序）：

   🥇 1st: QWEN @ λ=0.7     → 0.5967 ⭐⭐⭐
   🥈 2nd: GPT  @ λ=1.0     → 0.5952 ⭐⭐
   🥉 3rd: LLAMA @ λ=0.5    → 0.5919 ⭐

   差异: 仅 0.0048 (< 1%)
   结论: 三种backbone在最优配置下性能几乎相同

记忆机制影响排名（λ=0 → 最优λ的提升）：

   🥇 1st: LLAMA    +27.4%  (0.4514 → 0.5919)

   🥉 3rd: QWEN     +1.3%   (0.4714 → 0.5967)
   
   结论: LLAMA和GPT对记忆机制高度敏感
         QWEN本身已具备较强的上下文理解能力
